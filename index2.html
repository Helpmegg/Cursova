Ввід вивід в програмі на Python

Для виводу інформації в Python 3 використовується оператор print() , в дужках передається ше що треба вивести

print("Hello,world!")

Для отримання оператор input()

s = input()

s - це зміна в яку запишеться стрічка з терміналу.

Перетворення типів зміних

По стандарту оператор input() повертає стрічку , яку можливо перетворити у інші типи данних str() - стрічка float() - дробне число int() - ціле число Для перетворення в дужкає передається значення , а повертається результат Приклад виведенння числа яке було введене.

a = int(input()) print(a)

Арифметичні операції

a3 - результат a1 - оперант один a2 - оперант два

Додавання

a3 = a1 + a2

Віднімання

a3 = a1 - a2

Множення

a3 = a1 * a2

Ділення

a3 = a1 / a2

Степінь

a3 = a1 ** a2

Ціла частина ділення (Для цілих чисел)

a3 = a1 // a2

Остача частина ділення (Для цілих чисел)

a3 = a1 % a2

If - і інша опертори

If дозволяє перевіряти умови , і змінювати поведінку програми синтаксис if:

if a<18 : print("Привіт хлопчик!") elif a==18 : print("А ти виріс!") else: print("Ну чо повнолітній!")

Можливі умови: == рівне < менше > більше <= менше або рівне >= більше або рівне != не дорівнює Логічні оператори: or - або and - і

Цикли в Python

Цикл White

повторюється поки умова правдива

while(Умова для провження) тіло циклу

Приклад роботи while

while(Живий): пиши код

Цикл for

Цикл while продовжує виконуватися доти, доки умова залишається істинною. Якщо необхідно виконати блок коду лише визначену (відому) кількість разів, то використовують цикл for і функцію range(). Синтаксис використання цього циклу і цієї функції записується так:

for {назва зміної} in range(кінцеве значення): тіло циклу

cars = ['Alfa Romeo', 'Volvo', 'Lamborghini', 'BMW', 'Toyota']

for a in range(len()): print(x[a])
