Завдання «Шнурки»

Умова

Взуттєва фабрика збирається почати випуск елітної моделі черевик. Дірочки для шнурівки будуть розташовані в два ряди, відстань між рядами одно a, а відстань між дірочками в ряду b. Кількість дірочок в кожному ряду одно N. Шнуровка повинна відбуватися елітним способом "наверх, по горизонталі в інший ряд, наверх, по горизонталі і т.д." (Див. Рисунок). Крім того, щоб шнурки можна було зав'язати елітним бантиком, довжина вільного кінця шнурка повинна бути l. Яка повинна бути довжина шнурка для цих черевиків?

Рішення

a = int(input()) b = int(input()) L = int(input()) N = int(input()) print(2 * L + (2 * N - 1) * a + 2 * (N - 1) * b)

Задача «Шоколадка»

Умова

Шоколадка має вигляд прямокутника, розділеного на n × m часточок. Шоколадку можна один раз розламати по прямій на дві частини. Визначте, чи можна таким чином відламати від шоколадки частина, що складається рівно з k часточок. Програма отримує на вхід три числа: n, m, k і повинна вивести YES або NO.

Рішення

n = int(input()) m = int(input()) k = int(input()) if k < n * m and ((k % n == 0) or (k % m == 0)): print('YES') else: print('NO')

Умова

Вклад у банку становить x рублів. Щорічно він збільшується на p відсотків, після чого дрібна частина копійок відкидається. Визначте, через скільки років внесок складе не менше y рублів. Вираз «дрібна частина копійок відкидається» означає, що якщо у вас виявилося 123.4567 рублів, т. Е. 123 рубля і 45.67 копійок, то після округлення у вас вийде 123 рубля і 45 копійок, тобто 123.45 рублів. Програма отримує на вхід три натуральних числа: x, p, y і повинна вивести одне ціле число.

Задача «Унікальні елементи»

Умова

Даний список. Виведіть ті його елементи, які зустрічаються в списку тільки один раз. Елементи потрібно виводити в тому порядку, в якому вони зустрічаються в списку.

Рішення

a = [int(s) for s in input().split()] for i in range(len(a)): for j in range(len(a)): if i != j and a[i] == a[j]: break else: print(a[i], end=' ')

Задача «Кількість співпадаючих пар»

Умова

Даний список чисел. Порахуйте, скільки в ньому пар елементів, рівних один одному. Вважається, що будь-які два елементи, рівні один одному утворюють одну пару, яку необхідно порахувати.

Рішення

a = [int(s) for s in input().split()] counter = 0 for i in range(len(a)): for j in range(i + 1, len(a)): if a[i] == a[j]: counter += 1 print(counter)

index_of_max = 0 a = [int(i) for i in input().split()] for i in range(1, len(a)): if a[i] > a[index_of_max]: index_of_max = i print(a[index_of_max], index_of_max)

Задача «Числа Фибоначчі»

Умова

Напишіть функцію fib (n), яка за даним цілому неотрицательную n повертає n-e число Фібоначчі. У цьому завданні не можна використовувати цикли - використовуйте рекурсию.

Рішення

def fib(n): if n == 1 or n == 2: return 1 else: return fib(n - 1) + fib(n - 2) print(fib(int(input())))

Задача «Зведення в ступінь»

Умова

Дано дійсне позитивне число a і ціле невід'ємне число n. Обчисліть an не використовуючи цикли, зведення в ступінь через ** і функцію math.pow (), а використовуючи рекурентне співвідношення an = a⋅an-1.
Рішення оформите у вигляді функції power (a, n).

Рішення

def power(a, n): if n == 0: return 1 else: return a * power(a, n - 1) print(power(float(input()), int(input())))
