Інструкція по скачуваню і насройці Python 3

Щоб почати працювати з Python 3, вам потрібно отримати доступ до інтерпретатора Python. Існує кілька загальних способів зробити це:
Python можна отримати на сайті Python Software Foundation python.org. Як правило, це означає завантаження потрібно інсталяційний файл для вашої операційної системи і запуску його на вашому комп'ютері.
Деякі операційні системи, особливо Linux, надають менеджер пакетів, який можна запустити для установки Python.
Для macOS, кращий спосіб встановити Python 3 включає в себе установку менеджера пакетів під назвою Homebrew. Ви побачите, як це робити у відповідному розділі статті.
В якості альтернативи, існує ряд сайтів, які дозволяють вам отримати доступ до інтерпретатора Python онлайн, без необхідності в установки чогось на вашому комп'ютері. Перш ніж почати вивчати основи мови програмування Python ви повинні мати стабільну версію інтерпретатора Python.

Скачування Python на Windows

Імовірність того, що на вашій системі Windows заздалегідь встановлений Python - вкрай мала. На щастя, установка не вимагає більшого, ніж завантаження інсталяційний файл Python з сайту python.org і його запуску. Давайте розглянемо, як встановлювати Python 3 на Windows:


Крок 1: Завантаження інсталяційний файл Python 3

  • Відкриваємо вікно браузера і переходимо на сторінку Download для Windows на python.org


  • Під верхнім заголовком, де написано Python Releases for Windows, натискаємо на посилання до останньої версії Python 3.x.x. ;


  • Гортаємо вниз і вибираємо інсталяційний файл Windows x86-64 для 64-розрядної версії операційної системи або файл Windows x86 для 32-розрядної (Дивитися нижче).


32 або 64-бітний Python? У Windows ви можете вибрати як 32-бітний, так і 64-бітний інсталяційний файл. Розглянемо, в чому різниця між ними:


Якщо у вас система з 32-бітовим процесором, тоді вам потрібно вибирати 32-бітний інсталяційний файл.
Для 64-бітних систем, кожен інсталяційний файл може працювати в більшості випадків. 32-бітна версія використовує менше пам'яті, проте 64-бітна версія працює краще для додатків з інтенсивними обчисленнями.
Якщо ви не впевнені в тому, скільки разів вибрати, вибирайте 64-бітну.
Зверніть увагу: пам'ятайте, що якщо ви зрозуміли, що цей вибір неправильний, і хочете перейти до іншої версії Python, ви можете просто видалити Python і потім переустановити його, скачавши інший інсталяційний файл з python.org.

      Скачування Python 3 на Linux

      Імовірність того, що дистрибутив Linux містить встановлений Python набагато вище, але це не означає, що у вас встановлена ​​остання версія, і це може бути Python 2 а не Python 3.
      Щоб дізнатися, яка версія встановлена ​​у вас, відкрийте вікно терміналу і спробуйте виконати наступні команди:

      • python -version

      • python2 -version

      • python3 -version

      Одна або кілька цих команд повинні вказати на поточну версію:

      $ Python3 --version


      Python 3.6.5


      Або

      sudo apt install python

      Якщо зазначена версія є Python 2.x.x або версією Python 3, яка не є останньою, тоді вам може знадобитися остання версія. Процедура вирішення цього питання залежить від того, з яким дистрибутивом Linux ви працюєте.


      Скачування Python 3 на Mac OS X

      Поточна версія macOS включає в себе Python 2.7, проте його спокійно можна вважати застарілим кілька місяців тому. До речі, в даній статті використовується тільки Python 3, так що давайте спробуємо оновитися!
      Кращий спосіб установки Python 3 на macOS - це використовувати менеджер пакетів Homebrew. Цей підхід також рекомендуються в різних довідниках, такий як гайд до Python від Hitchhiker.
